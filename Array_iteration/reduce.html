JavaScript Arrays

The reduce() Method

Find the sum of all numbers in an array:

The reduce() method runs a function on each array element to produce (reduce it to) a single value. The reduce() method works from left-to-right in the array.

Note that the function takes 4 arguments: The total (the initial value / previously returned value) The item value The item index The array itself
